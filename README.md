# Chunk Pregenerator

A simple chunk pre-generation mod that generates chunks (maps) incrementally in the background while minimising server load.<br>
This aims to prevent asynchronous crashes during multiplayer sessions.<br><br>
サーバーへの負荷を抑えつつ、バックグラウンドで少しずつチャンク（マップ）を生成していくシンプルなチャンク事前生成MODです。<br>
マルチプレイ時の非同期クラッシュの回避を目的としています。
